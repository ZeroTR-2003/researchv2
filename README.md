## 🎯 Why This Project Matters

Namibia’s tourism industry is growing, but many potential visitors cannot physically explore landmarks before planning their trips. **Ascend Windhoek** bridges this gap by combining **3D exterior models** and **360° interior tours** into one seamless web experience.

This project demonstrates:
- **Full‑stack proficiency** with Next.js and TypeScript
- **3D graphics integration** using Three.js and React Three Fiber
- **Responsive, mobile‑first design** that works on phones and desktops
- **Performance optimization** for smooth 3D rendering
- **Real‑world problem‑solving** in cultural preservation and tourism

---

## ✨ Key Features

| Category | Details |
|----------|--------|
| 🎮 **Interactive 3D Models** | High‑quality GLB models with orbit controls, auto‑rotation, and camera presets |
| 🌍 **360° Virtual Tours** | Panoramic interior views with device motion support (gyroscope on mobile) |
| 🎨 **Premium Design** | Dark mode with persistence, luxury travel‑inspired UI, fully responsive |
| 🗺️ **Rich Information** | Interactive maps, quick facts, photo galleries, historical context |

### 🏛️ Featured Landmarks

1. **🕍 Christuskirche** – Historic Lutheran church (1907–1910)
2. **⚔️ Heroes' Acre** – National monument (2002)
3. **🏛️ Windhoek City Museum** – Cultural heritage centre
